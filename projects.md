# Project Descriptions
## Intellectus
Hybrid Knowledge Retrieval Platform for FinTech Research
Architected and deployed a dual-module information retrieval system
for FinTech industry combining graph-based and semantic search
capabilities that increased query accuracy by 40% over traditional
methods. Engineered an innovative architecture leveraging Neo4j
knowledge graphs for entity relationships and MongoDB for vector
embeddings, enabling intelligent querying across both structured and
unstructured financial content. Implemented LangChain orchestration
for context-aware responses, resulting in a 35% improvement in answer
relevance for complex financial queries.
Apr 2025 – May 2025
Technologies: Python, LangChain, Neo4j, MongoDB, Redis, gRPC,
Semantic Search, Knowledge Graphs

## Doc Detector and Classifier
Lightweight Document Analysis System for FinTech
Developed a high-efficiency document analysis system for FinTech
industry that processes multiple identity document types with 98% field
detection accuracy. Created and curated a custom-labeled dataset of
5,000+ financial documents, then trained a lightweight YOLOv8n model
optimized for edge deployment, reducing computational requirements
by 70% compared to standard solutions. Implemented a modular pipeline
supporting real-time inference for resource-constrained financial service
environments.
Oct 2024 – Nov 2024
Technologies: Python, YOLOv8n (Ultralytics), Roboflow, LabelImg,
PyTorch, OpenCV, FastAPI
